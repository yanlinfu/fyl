首页界面index.html.路径为:http://localhost/1905/fyl/index.html
进入首页之后,右上角登陆注册按钮.
登陆login.html.登陆直接路径:http://localhost/1905/fyl/login.html
若没有注册账号则需要先注册,登陆界面底部有免费注册按钮,点击之后进入注册页面,也可直接注册.
注册:register.html.直接路径为:http://localhost/1905/fyl/register.html.跳转至首页.
首页的推荐商品点击任意商品或者点击右上角更多之后将跳转至商品列表.
商品列表list.html,路径http://localhost/1905/fyl/list.html.
进入商品列表页面,点击详情,即可看到商品的详细介绍,
详情页:details.直接路径为http://localhost/1905/fyl/details.html
进入详情页,即可加入购物车.
购物车:car.html.直接路径:http://localhost/1905/fyl/car.html.加入购物车进入清算页面,即可对商品的数量进行加减和删除商品.
若想继续选择商品,可点击左上角继续购物.进入商品列表页面.
